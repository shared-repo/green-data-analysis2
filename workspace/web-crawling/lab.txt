1. crawling 1
    네이버 뉴스 -> IT/과학 -> IT 일반 영역
    최신 10 페이지의 목록 조회
    조회 항목 : 제목, 링크URL(a태그의 href 속성값)
    조회결과는 리스트에 저장
    다음 페이지 요청 전에 반드시 3초 이상의 interval 두어야 합니다.

2. crawling 2
   crawling 1의 조회 결과를 반복하면서 링크URL을 요청하고 내용을 crawling
   조회 결과는 리스트에 저장
   각 요청 사이에 1초 이상의 interval 두기

3. 네이버 뉴스외의 다른 사이트 crawling 선택 가능

   예 ) https://dzone.com/ai-ml
